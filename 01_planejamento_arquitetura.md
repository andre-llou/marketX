# Etapa 1 - Planejamento e Arquitetura

## Objetivo

Criar um marketplace estilo OLX focado em oportunidades de investimento.

## Stack Recomendada

### Frontend

-   Next.js
-   TailwindCSS
-   Axios

### Backend

-   Node.js
-   Express
-   Prisma ORM
-   PostgreSQL

### Upload de imagens

-   Cloudinary ou AWS S3

### Autenticação

-   JWT

## Estrutura do Projeto

/marketplace ├── frontend └── backend
